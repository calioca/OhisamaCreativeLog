# 🌞 Sunny Creative Log

> 創ることを、少しずつ積み重ねる。

創作活動を記録し、振り返るための個人用Webアプリです。

日々の創作ログを記録し、
「今日は何を作ったか」
「どれくらい創作したか」
を気軽に振り返ることを目的としています。

---

## ✨ 現在の機能

- 創作ログの登録
- 創作ログの編集・削除
- 作品ごとの管理
- 作品名の編集
- ログのない作品の削除
- 今日の創作一覧表示
- 今日の文字数集計
- localStorageによるデータ保存
