# ☀️ Sunny Creative Log

創作の「今日」と「積み重ね」を記録する、シンプルな創作ログアプリです。

noteやしずかなインターネットなど、複数の創作活動を作品ごとに記録できます。

GitHub Pages: 

---

## 主な機能

- ✍️ 創作ログの記録
- 📚 作品ごとの管理
- 🌞 今日の創作集計
- 📅 今月の創作集計
- 📖 作品詳細画面
- 🗑️ 最後のログ削除時に作品も自動削除

---

- CSS

- LocalStorage
